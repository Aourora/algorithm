# 链表

1. 数据结构
    - 链表是一种线性数据结构，它是一种用"指针"将零散的内存空间串联起来数据结构
2. 链表的优缺点
    - 优点：高效的插入和删除操作
    - 缺点：随机访问的时间复杂度为 O(n)
